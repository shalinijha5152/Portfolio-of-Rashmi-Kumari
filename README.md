# Professional Portfolio | Equity Research Analyst

Welcome to my personal portfolio website repository. This platform serves as a centralized hub to showcase my background in fundamental equity analysis, financial modeling, and investment writing across US and Indian markets.

## 🔗 Live Demo
You can view the live, fully interactive website here: https://portfolio-of-rashmi-kumari.onrender.com

---

## 🎨 Design & Aesthetic
The website features a highly professional, modern, and clean editorial design utilizing a crisp white canvas with sophisticated soft berry pink and slate highlights. It is fully responsive across desktop, tablet, and mobile displays.

---

## 🛠️ Tech Stack & Key Features

- **Frontend Architecture:** Clean, semantic HTML5 and modern CSS3 custom layouts.
- **Dynamic Interactions:** Vanilla JavaScript implementation for an automatic home-screen typing effect and custom page scrolling triggers.
- **Entrance Animations:** Smooth, sequential stagger delays that animate layout text boxes and credential cards one-by-one as the user scrolls.
- **Integrated Email Engine:** Powered via Formspree to seamlessly route professional inquiries from the contact form directly to my inbox.

---

## 📂 Portfolio Structure & Sections

- **Home:** A premium hero section featuring an animated text layout typist, professional headshot, and a direct cloud-download asset action for my resume.
- **About:** A high-level overview of my core professional philosophy, detailing how I transform complex macroeconomic data into actionable corporate investment insights.
- **Work Experience:** An interactive, vertically aligned timeline tracing my analytical contributions across organizations like PredictRAM and Flair Tree.
- **Education:** Structured cards detailing my academic background from the University of Calcutta and IGNOU.
- **Published Research:** A crisp grid highlighting research, earnings previews, and investment commentaries featured on major financial networks including **NASDAQ, Yahoo Finance, and Barchart**.
- **Core Skills:** Clean categorical badge filters mapping out competencies from DCF Valuation & Comps to tools like FactSet, Advanced Excel, and Power BI.
- **Certifications:** A visual showcase displaying technical training milestones from Internshala and DataCamp.
- **Contact:** An operational communications portal styled with vector contextual iconography.
